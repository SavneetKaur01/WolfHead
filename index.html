WolfHead is a decentralized application platform designed to make apps usable on the web.

The network runs on a Proof-of-Stake (POS) consensus mechanism called Nightsade, which aims to offer scalability and stable fees.

WolfHeadUSD is the native utility token that is used for:

  • Fees for processing transaction and storing data
  • Running validator nodes on the network via staking WolfHeadUSD token
  • Used for governance votes to determine how network resources are allocated
